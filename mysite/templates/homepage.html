Welcome to My Project!

This is a showcase of my coding abilities and understanding of Django framework, containing four distinct applications: a Forum, a HIIT Workout Tracker, a Notes app, and a User Profile app. The goal of this project was to create an integrated platform to demonstrate my skills in developing full-stack applications.

Forum

A fully-featured forum where users can create, read, update, and delete posts. This also includes the ability for users to engage in discussions by leaving comments, demonstrating CRUD operations and database relationships.

HIIT Workout Timer

A High-Intensity Interval Training (HIIT) tracker, showcasing the development of a real-time, interactive tool that allows users to set and track their workout routines.

Notes

An application that allows users to keep track of important notes and reminders. Demonstrates a CRUD model with the additional complexity of user-defined reminders.

User Profile

An app that lets users manage their profiles. Users can change profile pictures, update personal details, and manage reminders. This section showcases user authentication and user-specific data handling.
